# Sales Data Analysis using Python & SQL

This project analyzes sales data to extract meaningful business insights.

## Tools Used
- Python
- Pandas
- SQL
- Matplotlib

## Features
- Sales data analysis
- Region-wise and category-wise sales
- Data visualization
- SQL queries for reporting

## Objective
To demonstrate data analysis and problem-solving skills using real-world data.
